Features
--------

* **Interactive & Beautiful**: Rich CLI interface with colors, progress bars, and interactive menus
* **User-friendly**: Comprehensive help system and intuitive command structure
* **Multi-platform**: Support for both Android and iOS development workflows
* **Asset Generation**: Automated icon and splash screen generation
* **Environment Health**: Built-in doctor command for environment diagnostics

Quick Start
-----------

Install Flow CLI:

.. code-block:: bash

   pip install flow-cli

Run the doctor command to check your environment:

.. code-block:: bash

   flow doctor

Get help for any command:

.. code-block:: bash

   flow --help
   flow android --help
   flow ios --help
